# Іскра ❤️ — сервіс знайомств

## Про проєкт

Іскра — це веб-сервіс знайомств, який допомагає користувачам створювати анкети, знаходити людей зі схожими інтересами та спілкуватися.

## Реалізований функціонал

- Реєстрація користувача
- Авторизація
- Створення анкети
- Перегляд анкет
- Система взаємних симпатій
- Чат між користувачами
- Користувацький інтерфейс

## Технології

- HTML5
- CSS3
- JavaScript
